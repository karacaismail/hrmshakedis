# hrmshakedis — Asgari İşçilik & Hakediş Uyum Platformu

Frappe v16 + Frappe HR çekirdeği üzerinde, tek geliştiriciyle inşa edilen **headless** SGK asgari işçilik ve hakediş uyum ürününün stratejik yol haritası.

**Canlı rapor:** https://karacaismail.github.io/hrmshakedis/

## İçerik

Rapor tek sayfadır (`index.html`) ve şunları kapsar: vizyon/misyon/değerler, ürün geliştirme kültürü (vibe engineering), mevzuat çerçevesi (5510 md.85, SSİY md.110-112, SGK Genelgesi 2026/16), headless mimari, SMART hedefler, sprint bazlı yol haritası, MoSCoW backlog dağılımı, risk matrisi ve unk-unk yönetim çerçevesi, gap analizi, test stratejisi (altın dosya yaklaşımı) ve kaynakça.

## Teknoloji

Tailwind CSS (CDN) · Apache ECharts 5 · Phosphor Icons · Roboto. Derleme adımı yoktur; GitHub Pages doğrudan sunar.

## Notlar

- Müşteri adı ticari gizlilik gereği raporda anonimleştirilmiştir ("Kurumsal Müşteri").
- Projelendirme çalışma kitabı (Excel) müşteri verisi içerdiği için bu depoda YER ALMAZ; ayrı kanaldan paylaşılır.
- Rapor bilgilendirme amaçlıdır; resmi SGK hesabı yerine geçmez.
